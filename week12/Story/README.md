# Smart Library (JavaScript)

A clean JavaScript project for managing a library with validation,
borrowing, expiration logic, and purchase planning.

## Features
- Book validation & normalization
- Immutable library updates
- Low-copy & expiration detection
- Purchase list generation
- Searching & summaries
- Borrowing / returning
- Overdue reports
- Popular books
- Duplicate merging

## Tech
- JavaScript (Node.js)
- Jest for unit testing

## Install & Test
```bash
npm install
npm test
